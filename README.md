# IOgame Project

Spring Boot 기반 IO 게임 웹 애플리케이션  
웹 게임 시작 전 메인 페이지 [회원가입, 로그인, 랭킹, 게시판] UI, UX 및 기능 구현


## 주요 기능

- 회원가입 및 로그인 기능
- 사용자 랭킹 페이지
- 게시판 작성 및 목록 조회
- 로그인 시 댓글 기능 사용 가능
- 사용자 프로필 보기
- 메인 페이지 및 게임 진입 구조 구성


## 사용 기술

|  분야       | 기술 스택                            |
|------------|-------------------------------------|
| Language   | Java 17                             |
| Backend    | Spring Boot, Spring MVC             |
| Build Tool | Maven                               |
| Database   | MariaDB                             |
| View       | Thymeleaf, HTML, CSS                |
